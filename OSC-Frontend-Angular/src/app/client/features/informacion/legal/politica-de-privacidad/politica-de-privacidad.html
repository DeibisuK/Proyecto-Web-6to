2. Finalidad del Tratamiento de Datos

Los datos personales recopilados en OSC Sport Center se utilizan exclusivamente con fines académicos:

  • Simular el funcionamiento de un sistema de reservas deportivas y tienda online.
  • Evaluar la interacción de usuarios en un entorno deportivo controlado.
  • Demostrar habilidades técnicas en desarrollo web y gestión de bases de datos.
info

Nota:

  • Los datos ingresados (nombre, correo, documento de identificación) son ficticios o de prueba.
  • No se realizan transacciones comerciales reales ni envío de comunicaciones promocionales.

3. Legitimación del Tratamiento

  • Base legal: Ejecución de un proyecto académico autorizado por la Universidad Técnica de Machala.
  • Consentimiento del usuario: Al interactuar con la plataforma, aceptas que los datos simulados se usen para fines educativos.

4. Datos Recopilados

4.1. Tipos de Datos

  • Datos simulados: Nombre, correo electrónico, tipo de documento de identidad.
  • Datos técnicos: Dirección IP, navegador utilizado, sistema operativo (recopilados automáticamente para análisis de funcionamiento).

4.2. Procedencia

Todos los datos provienen directamente del usuario durante la simulación de reservas y compras.

5. Almacenamiento y Seguridad

Plazo de conservación: Los datos se eliminarán al finalizar la evaluación del proyecto.

Herramientas: Bases de datos locales o servicios académicos (ejemplo: Firebase, MySQL).

Medidas de seguridad: Encriptación básica y acceso restringido a los creadores del proyecto.

6. Cesión de Datos

No se comparten datos con terceros, empresas o entidades públicas.

Excepción: Si se usan herramientas académicas (ejemplo: GitHub Pages, Firebase), estos servicios pueden almacenar datos bajo sus propias políticas.

7. Derechos del Usuario

Aunque los datos son simulados, en el contexto académico:

  • Acceso y rectificación: Puedes solicitar ver o corregir datos ficticios asociados a pruebas.
  • Eliminación: Enviar una solicitud a [correo académico] para borrar información de prueba.
  • Limitación u oposición: Contactar a los creadores si deseas restringir el uso de datos simulados.

8. Transferencias Internacionales

No aplica: OSC Sport Center opera como un prototipo local.

Si se usan servicios en la nube (ejemplo: Firebase), los datos se rigen por las políticas de dichas plataformas.

9. Modificaciones

Los creadores pueden actualizar esta política para reflejar cambios en el proyecto.

Se notificarán cambios mediante un aviso en la plataforma.
